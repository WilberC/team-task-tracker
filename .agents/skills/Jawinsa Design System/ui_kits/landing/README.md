# Jawinsa Landing — UI Kit

A hi-fi recreation of the Jawinsa Repara landing page. Built with React 18 + Babel (no build step).

## Files
- `index.html` — assembled landing (open this)
- `Navbar.jsx`
- `Hero.jsx`
- `ServiceLines.jsx`
- `WhyJawinsa.jsx`
- `News.jsx`
- `Contact.jsx`
- `Footer.jsx`
- `kit.css` — kit-local styles + photo placeholders

## Sections (in order)
1. Hero — signature parallax photo + line-by-line headline reveal + amber underline draw
2. Service lines — 4 cards: automotriz, comercial, industrial, repuestos
3. Why Jawinsa — stats block + three trust pillars
4. News — three blog entries
5. Contact — quote form + WhatsApp + map placeholder
6. Footer

## Placeholders
All photography is rendered as warm graphite gradients (clearly labelled). Replace with real workshop photos.
